# MBTI by Movies

A personality analysis tool that infers your MBTI type based on your movie preferences.

## Description

This project uses artificial intelligence to analyze your movie preferences (liked and disliked movies) and infer your MBTI personality type. It combines Jungian psychology, MBTI theory, and film aesthetics to provide deep insights into your cognitive function preferences.

## Features

- Analyzes movie preferences to infer MBTI type
- Provides detailed personality reports based on film aesthetics
- Uses psychological frameworks to connect movie choices with cognitive functions
- Generates personalized movie recommendations
